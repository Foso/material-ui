CssBaseline API

The API documentation of the CssBaseline React component. Learn more about the props and the CSS customization points.

Import

import CssBaseline from '@material-ui/core/CssBaseline';
// or
import { CssBaseline } from '@material-ui/core';

You can learn more about the difference by reading this guide.

Kickstart an elegant, consistent, and simple baseline to build upon.

Props

Name Type Default Description
children node null You can wrap a node.
classes object Override or extend the styles applied to the component. See CSS API below for more details.

The component cannot hold a ref.

CSS

  • Style sheet name: MuiCssBaseline.
  • Style sheet details:
Rule name Global class Description
@global Apply global styles.

You can override the style of the component thanks to one of these customization points:

  • With a rule name of the classes object prop.
  • With a global class name.
  • With a theme and an overrides property.

If that's not sufficient, you can check the implementation of the component for more detail.
